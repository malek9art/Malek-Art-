## 🇸🇦 عن المشروع
**مالك أرت (Malek Art)** هو موقع إلكتروني تعريفي ومعرض أعمال إبداعي مخصص للمصممين ومطوري الويب. يتميز الموقع بتصميمه العصري القائم على أسلوب **الـ Glassmorphism** والتأثيرات الحركية الغامرة لتمنح العميل تجربة فريدة.

يحتوي الموقع على **لوحة تحكم كاملة لإدارة الـ CMS** لتعديل المشاريع والخدمات والنصوص بشكل فوري، بالإضافة إلى **نظام مستشار التخطيط الذكي (AI Matchmaker)** المدعوم بنموذج **Gemini 3.5 Flash** لتقديم خطط عمل تسويقية وتقنية شاملة للعملاء ومطابقتها وتوصيتها بمنتجات سابقة.

---

## 🌐 Project Overview (En)
**Malek Art** is a premium, high-converting digital portfolio built for creative designers and web developers. Featuring a state-of-the-art visual architecture (Glassmorphism, deep shadows, and custom ambient glowing structures), it establishes a solid gold standard in digital branding. 

Beyond standard exhibitions, it integrates an **Intelligent Project Advisor (AI Matchmaker)** powered by **Gemini 3.5**, a full-scale **Bespoke CMS Admin Panel** to manipulate content, and a responsive **Client Design Proposal Queue** that bridges interactive planning with real-time operations.

## ✨ الميزات والخصائص / Core Features

### 1. 🤖 مستشار التخطيط الذكي (AI Matchmaker)
* **استبيان مرئي متكامل**: يضم تحديد نوع الخدمة، التوجه البصري والأمور الفنية المطلوبة للعميل.
* **تحليل فوري دقيق**: يتصل بـ **Gemini API** لتحويل أفكار وميزانية العميل المكتوبة بأسلوب سطحي إلى مستند تخطيط فني مع جدول زمني دقيق وأدوات مقترحة مجاناً.
* **مطابقة ذكية للنماذج ومقارنتها**: يوصي العميل تلقائياً بمشروع شبيه من قاعدة بيانات أعمال مالك بناء على فكرته.
* **نظام طوارئ ذكي (Fallback Mode)**: في حال عدم إدخال مفتاح الذكاء الاصطناعي، يتم تشغيل خوارزميات محلية ذكية ومحاكية تقوم بصياغة التقرير بشكل استجابي ومطابق للاختيارات.

### 2. 🛡️ بوابة الإدارة الذكية والـ CMS
* **لوحة تحكم محمية بكلمة مرور**: للدخول وتحديث كافة بيانات الموقع.
* **إدارة حية للمشاريع والخدمات والمهارات**: إضافة، مسح، وتحديث المشاريع وحفظها في الذاكرة.
* **محرر نصوص وهوية العرض**: تعديل عبارات الافتتاحية والنبذة الشخصية والصور والروابط ومستشار الذكاء بشكل حي ومباشر دون المساس بالأكواد البرمجية.
* **لوحة طلبات التصميم وحالة معالجتها**: عرض وتتبع طلبات التخطيط والمقترحات التي أرسلها العملاء واختياراتهم الشخصية مع إمكانية تعليمها كـ "مكتملة" أو "تحت الإجراء" أو حذفها.

### 3. 🗺️ تصميم تفاعلي متميز وتجربة مستخدم مذهلة
* **دعم اللغتين (العربية والإنجليزية)**: توافق تام وتلقائي مع اتجاه الحروف LTR/RTL والخطوط المناسبة للهيكلية.
* **متحركات سلسة**: باستخدام مكتبة **Motion** لخلق تجارب تصفح هادئة وغامرة وممتازة للأجهزة المحمولة.
* **مظهر متكامل وراحة للعين**: ألوان منتقاة بعناية وتأثيرات زجاج تليق بالمطورين والعملاء المتميزين.

## 🛠️ الأدوات والبنية التقنية / Tech Stack

* **الواجهات والتصميم (Frontend)**: React 19 + TypeScript + Vite.
* **التنسيق (Styling)**: Tailwind CSS v4.
* **الحركية (Animations)**: Motion.
* **الخلفية (Backend)**: Express Server لتفادي كشف الـ API keys في المتصفح.
* **الذكاء الاصطناعي (AI engine)**: الحزمة الحديثة الرسمية لشركة جوجل `@google/genai` المدعومة بنموذج `gemini-3.5-flash`.
* **الأيقونات (Vector Icons)**: Lucide React.

## ⚙️ طريقة التشغيل المحلي / Local Setup Guidelines

لتشغيل المشروع على جهازك الشخصي والاستعداد لمزامنته مع مستودع غيث هاب الخاص بك بمرونة:

### 1. استنساخ المستودع أو تنزيله / Clone or Download
قم بتحميل ملفات المشروع وفتحها في محرر الأكواد المفضل لديك (مثل VS Code).

### 2. تثبيت الحزم البرمجية / Install Dependencies
افتح سطر الأوامر في مجلد المشروع وقم بتشغيل الأمر التالي لتنزيل جميع حزم التطبيق:
```bash
npm install
```

### 3. إعداد متغيرات المتصفح والذكاء الاصطناعي / Configuration `.env`
قم بنسخ ملف `.env.example` وتسميته بـ `.env`:
```bash
cp .env.example .env
```
افتح ملف `.env` وقم بإرفاق مفتاح الـ **Gemini API Key** الخاص بك ليقوم التطبيق بالاتصال الحقيقي:
```env
# احصل على مفتاح مجاني من Google AI Studio
GEMINI_API_KEY=YOUR_GEMINI_API_KEY_HERE
```

*ملاحظة: إذا تركت المفتاح فارغاً، سيتعرف التطبيق على ذلك تلقائياً وسيقوم بتشغيل محاكي مستشار التخطيط الداخلي (Fallback Simulation Mode) لضمان عدم توقف الخدمات أمام الزوار.*

### 4. تشغيل خادم التطوير / Launch Development Server
قم بتشغيل الأمر التالي لتبدأ الأكواد بالعمل محلياً:
```bash
npm run dev
```
سيكون الموقع متاحاً على الرابط التالي:
👉 **`http://localhost:3000`**

## 🚀 طريقة النشر / Production Deployment

المشروع مصمم ليكون جاهزاً لإطلاقه فوراً على السيرفرات السحابية (مثل Cloud Run أو Render او Vercel):

### البناء والضغط للمشروع / Compilation & Production Build
يقوم أمر البناء بتهيئة وضغط ملفات المتصفح، وترجمة السيرفر الخلفي إلى ملف مدمج ذاتياً داخل مجلد الـ `dist`:
```bash
npm run build
```

### بدء تشغيل النسخة المستقرة / Production Spin up
```bash
npm run start
```
